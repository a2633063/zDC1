## 特性

本固件使用斐讯DC1排插硬件为基础,实现以下功能:

- [x] 4个USB充电
- [x] 按键控制所有插口通断
- [x] 控制每个接口独立开关
- [x] 每个接口拥有独立的5组定时开关
- [x] ota在线升级
- [x] 无服务器时使用UDP通信
- [x] MQTT服务器连接控制
- [x] 通过mqtt连入homeassistant
- [x] app实时显示功率/电压/电流(不做任何精度保证)

## 开始

整体流程如下:拆开DC1,将固件/烧录器/pc互相连接,在pc运行烧录软件进行烧录,烧录固件.

烧录完成后,首次使用前配对网络并配置mqtt服务器,之后就可以使用了.
